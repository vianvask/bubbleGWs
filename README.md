# bubbleGWs
C++ code that computes the gravitational wave spectra from strongly supercooled first-order phase transitions in the thin-wall limit accounting for the cosmic expansion.

Please cite [2511.15687](https://arxiv.org/abs/2511.15687) if you use this code. The data used in [2511.15687](https://arxiv.org/abs/2511.15687), generated with this code, is available at [Zenode](https://zenodo.org/records/17726613).
